# 식사 약속 조율 앱 명세 평탄화
1. 앱은 식사 약속 생성과 공유, 가용 시간 조율을 위한 반응형 웹 애플리케이션이다.
2. 사용자는 약속마다 입력하는 닉네임으로만 식별되며 개인 정보는 수집하지 않는다.
3. 생성자는 약속 제목, 간단한 설명, 시간 슬롯 규칙 세트를 입력해 약속을 만든다.
4. 약속 생성 시 공유용 링크가 자동 발급되며 초대 코드는 제공되지 않는다.
5. 생성자는 발급된 공유 링크를 복사해 원하는 채널에 직접 붙여넣는 방식으로 배포한다.
6. 시간 슬롯은 입력된 규칙에 따라 자동 생성되며 구체적인 규칙 옵션은 추후에 결정한다.
7. 자동 생성된 시간 슬롯은 사용자에 의해 수동 추가 또는 삭제할 수 없다.
8. 사용자는 공유 링크로 접근해 닉네임을 입력하고 약속에 참여한다.
9. 참여자는 필요할 경우 본인 확인용 짧은 비밀번호를 선택적으로 설정할 수 있다.
10. 참여자는 참여 가능한 시간 슬롯을 선택하여 표시하고 선택하지 않은 슬롯은 불가능으로 처리된다.
11. 참여자는 자신의 선택을 언제든지 수정하거나 갱신할 수 있다.
12. 앱은 각 시간 슬롯별 선택한 참여자 수와 응답률을 시각화한다.
13. 참여자는 자신의 선택과 전체 현황을 동시에 확인할 수 있다.
14. 앱은 확정, 잠금 등 추가 상태를 제공하지 않고 가용성 공유 수준에서 약속 조율을 지원한다.
15. 저장 데이터는 약속 정보, 시간 슬롯, 선택된 슬롯, 닉네임, 선택적 짧은 비밀번호에 한정된다.
16. 세션 토큰과 같은 추가 식별 정보는 저장하지 않는다.
17. 이메일이나 전화번호 등 개인 식별 정보는 수집하지 않는다.
18. 데이터 자동 삭제 기능은 제공되지 않으며 보존 정책은 향후 결정된다.
19. 로그인 없이 약속을 생성하고 참여할 수 있도록 최소 입력 흐름을 유지한다.
20. 인터페이스는 모바일 사용성을 우선 고려해 단순하고 직관적으로 설계된다.
21. 고급 보안 절차보다 접근성과 사용 편의성을 우선하되 공유 링크는 난수화한다.
