# 식사 약속 조율 앱 명세 평탄화
1. 앱은 식사 약속 생성, 공유, 시간 조율을 위한 반응형 웹 애플리케이션이다.
2. 사용자는 개인정보 대신 임시 닉네임이나 기기 세션으로 식별된다.
3. 생성자는 약속 제목, 설명, 식사 유형, 유효 기간, 규칙 세트를 입력해 약속을 만든다.
4. 약속 생성 시 참여 링크와 초대 코드가 자동 발급된다.
5. 생성자는 링크 복사, QR, 메시지 공유 등으로 약속을 배포한다.
6. 시간 슬롯은 요일 반복, 날짜 범위, 식사 시간대, 시간 간격 규칙으로 자동 생성된다.
7. 생성자는 자동 생성된 시간 슬롯을 수동으로 추가 또는 삭제할 수 있다.
8. 모든 시간 슬롯은 타임존 정보를 포함해 참여자에게 동일하게 제공된다.
9. 초대 링크나 코드로 들어온 사용자는 닉네임을 입력하고 약속에 참여한다.
10. 참여자는 각 시간 슬롯에 대해 가능, 불가능, 미정, 선호도를 표시할 수 있다.
11. 참여자는 자신의 응답을 언제든지 수정하거나 갱신할 수 있다.
12. 생성자는 참여자 목록을 확인하고 응답을 잠금 또는 삭제할 수 있다.
13. 앱은 슬롯별 가능 인원, 미정 인원, 응답률을 시각화한다.
14. 참여자는 개인 응답과 전체 현황을 동시에 확인한다.
15. 생성자는 가중치 기반 추천이나 수동 선택으로 확정 시간 슬롯을 결정한다.
16. 확정된 시간과 장소는 모든 참여자에게 표시된다.
17. 앱은 새로운 참여, 응답 수정, 확정 등 변경 사항을 알림과 선택적 푸시로 전달한다.
18. 알림 설정은 약속 단위 on/off 토글로 제공된다.
19. 저장 데이터는 닉네임, 응답 상태, 선택 코멘트 등 비식별 정보와 세션 토큰, 약속/슬롯/응답 데이터에 한정된다.
20. 이메일이나 전화번호 등 개인 식별 정보는 수집하지 않는다.
21. 약속 종료 30일 후 데이터는 자동 삭제된다.
22. 로그인 없이 약속 생성 및 참여가 가능하다.
23. 핵심 흐름은 생성, 공유, 응답, 확정을 3단계 이내 클릭으로 처리하도록 설계한다.
24. 보안은 링크 난수화와 세션 만료 등 기본 수준으로 유지하며 사용성을 우선한다.
25. 프런트엔드는 React 기반 SPA와 반응형 UI 라이브러리를 사용한다.
26. 백엔드는 서버리스 또는 경량 Node.js 서비스로 약속/슬롯/응답 API를 제공한다.
27. 실시간 현황 갱신을 위해 웹소켓 또는 짧은 주기 폴링을 사용한다.
28. 배포는 HTTPS 지원 관리형 플랫폼과 맞춤 도메인을 이용한다.
29. 향후 기능으로 캘린더 내보내기, 추천 알고리즘 고도화, 그룹 채팅을 고려한다.
